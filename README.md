# PW6
Навчальний Android-застосунок мовою Kotlin для розрахунку електричних навантажень цехової мережі методом упорядкованих діаграм.

## Опис
У межах практичної роботи №6 я реалізувала мобільний калькулятор для автоматизованого розрахунку електричних навантажень цехової мережі.

## Технології
- Kotlin  
- Android Studio
